是否有可行性？那就站在巨人的肩膀上，做毕业照人脸识别..<br>
做一个回忆册的APP，记录校园每一届毕业生的毕业照，然后利用人脸识别系统，每个人脸对应他的个人信息（姓名，性别，联系方式，专业），也可以方便毕业之后联系<br>
背景：灵感来源于小时候填的同学回忆录，但是纸质的可能只会不会经常打开，APP可以随时打开，建立并维系强大的社交网，在毕业后有需要还能联系。<br>
用户：所有中山大学南方学院毕业生<br>
API：微软人脸识别<br>
前期：收集应届毕业生的数据<br>
